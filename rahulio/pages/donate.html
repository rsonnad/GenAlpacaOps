Preserving a Living Tradition

Gungru Khangtsen is one of eleven monastic houses within Gaden Shartse Monastery in Mundgod, India — home to the exiled Tibetan monastic community since 1966. Our monks dedicate their lives to the study of Buddhist philosophy, debate, and meditation, preserving a tradition stretching back over six centuries. Through annual USA tours featuring sacred sand mandala creation, teachings, and traditional ceremonies, we share this wisdom with the world while raising essential funds for the monastery.

Google Pay (GPay): India's most popular UPI app. Scan the QR code or enter the UPI ID above. Instant transfer, zero fees for sender. Available on Android and iOS.

PhonePe: Second-largest UPI app. Same QR/UPI flow. Supports UPI Lite for small amounts (under ₹500) without PIN entry.

Paytm: Wallet + UPI hybrid. Pay from Paytm Wallet balance or linked bank account via UPI. Also supports Paytm Postpaid (buy now, pay later).

BHIM: Government's official UPI app by NPCI. Lightweight, works on basic smartphones. Direct bank-to-bank transfer with no intermediary.
What is UPI? Unified Payments Interface is India's real-time payment system, built by the National Payments Corporation of India (NPCI). It processes over 12 billion transactions per month with zero fees for person-to-person transfers. Works 24/7 including bank holidays. Over 300 bank apps support UPI.

Frontend
Pure HTML/CSS/JS — no React, no build step, no npm. Served from GitHub's global CDN (Fastly). Time-to-first-byte under 50ms worldwide. The entire page weighs under 50KB gzipped.
💳
Payment Processing
Razorpay handles PCI compliance so we never touch card numbers. Client-side tokenization converts sensitive data before it reaches our servers. Razorpay processes $100B+ annually for 10M+ businesses.
📱
UPI Integration
UPI QR codes generated dynamically with the exact donation amount pre-filled. Donor scans and confirms with their PIN. Works with 300+ bank apps. UPI processed 12.02 billion transactions worth $250B in January 2024.
🗃
Database
PostgreSQL via Supabase — the same database engine powering Instagram and Spotify. Row Level Security ensures donors only see their own data. Real-time subscriptions push updates to the admin dashboard instantly.
📧
Email Receipts
Branded HTML emails via Resend API (99.9% deliverability). Indian donors receive 80G tax exemption certificate number. International donors receive a valid tax-deduction receipt. Sent within 30 seconds of payment.
💬
SMS Confirmation
Telnyx carrier-grade SMS to 200+ countries. Indian donors get SMS in Hindi or English based on preference. Includes amount, transaction ID, and a thank-you blessing.
🔁
Recurring Billing
Razorpay Subscriptions API with automatic retry — if a monthly charge fails, it retries 3 times over 7 days before pausing. Donors can pause, resume, or cancel anytime via a self-service portal link in each receipt email.
💲
Multi-Currency
Real-time exchange rates from Razorpay. Donors see amounts in their local currency at checkout. Settlement in INR. Supported: INR, USD, EUR, GBP, CAD, AUD, SGD, JPY, HKD.
📝
80G Tax Receipts (India)
Automatic 80G tax exemption certificates. Section 80G of the Income Tax Act allows 50% deduction on donations to approved institutions. Unique certificate numbers generated and tracked per donor.
🇺🇸
501(c)(3) US Tax Deduction
US donors giving through the Gaden Shartse Cultural Foundation (EIN: 20-5126355) receive tax-deductible receipts. Deductible up to 60% of AGI for cash, 30% for appreciated property.
📜
FCRA Compliance
Valid FCRA registration from India's Ministry of Home Affairs permits receiving foreign donations. All foreign contributions reported quarterly. Full transparency — annual reports available on request.
🛡
Fraud Prevention
Multi-layer protection: Razorpay AI-powered fraud detection, 3D Secure card authentication, UPI PIN verification, webhook signature validation (HMAC-SHA256), and server-side amount verification. Client-side amounts never trusted.
🔒
Data Privacy
Donor data stored in Supabase with Row Level Security. No data shared with third parties. Anonymous donation option — name and email not stored if anonymous is checked. All data encrypted at rest (AES-256) and in transit (TLS 1.3).
🔗
Webhook-Driven
Payment confirmations via Razorpay webhooks trigger ledger updates, receipts, and notifications automatically. HMAC signature verification on every webhook to prevent spoofing.
🤖
AI Assistant
PAI-powered chatbot for donation questions, available in English, Hindi, and Tibetan. Answers property, scheduling, and donation queries via chat, voice, email, or Discord.
📈
Cost Tracking
All API calls logged to api_usage_log for full financial transparency. Per-vendor, per-category cost breakdowns on the admin dashboard. Every rupee accounted for.
💻
Admin Dashboard
Monastery administrators see real-time donation feeds, donor CRM, payment reconciliation, and financial reports. Built on the AlpacApps admin infrastructure. Mobile-responsive for access from anywhere.
☁
Zero Server Cost
Fully serverless architecture. Supabase Edge Functions handle all backend logic on demand. No servers to maintain, patch, or scale. Free tier covers thousands of donations per month.
Donation Data Flow
1 Donor clicks “Donate Now” — JS calls Supabase Edge Function to create a Razorpay Order
2 Razorpay Checkout modal opens (hosted by Razorpay — PCI compliant). Donor completes payment via card, UPI, netbanking, or wallet
3 Razorpay sends webhook to Supabase Edge Function — function verifies HMAC-SHA256 signature
4 Writes to: donations table + ledger table + api_usage_log for cost tracking
5 Triggers: receipt email via Resend (with 80G certificate number for Indian donors) + SMS confirmation via Telnyx
6 Updates admin dashboard in real-time via Supabase Realtime subscriptions
Monthly Recurring Flow
1 Razorpay Subscriptions API creates plan + subscription on first donation
2 Auto-charges on billing date each month — same webhook flow triggers receipts and ledger entries
3 Failed charge → 3 retries over 7 days → notification email to donor if still failing
4 Donor self-service portal link in each receipt for pause, resume, or cancel
