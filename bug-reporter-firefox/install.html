AlpacApps Bug Reporter

Firefox extension for reporting bugs with annotated screenshots

Installation

Android
Desktop
1

Install Firefox for Android

Make sure you have Firefox for Android installed (not Firefox Focus or Firefox Lite).

2

Download the extension

Tap the button below on your Android device to download the extension file.

Download for Firefox Android

If the download doesn't start, long-press the button and choose Save link.

3

Install the extension

Firefox should prompt you to install the add-on. Tap Add when asked.

If Firefox doesn't prompt automatically, open Firefox and go to about:addons, tap the gear icon, then tap Install Add-on From File and select the downloaded file.

4

Grant permissions

When prompted, tap Add to allow the extension to capture screenshots.

1

Download the extension

Download and unzip the extension folder to your computer.

Download from GitHub

After unzipping, the extension is in the bug-reporter-firefox folder.

2

Open Firefox debugging page

Type about:debugging#/runtime/this-firefox in your Firefox address bar and press Enter.

3

Load the extension

Click Load Temporary Add-on, then navigate to the bug-reporter-firefox folder and select the manifest.json file.

4

Find the extension

The extension icon will appear in your toolbar. Click it to open the bug reporter.

How to Use

1

Navigate to the page with the bug

Go to the page where you see the issue.

2

Open the bug reporter

Android: Tap the three-dot menu (⋮), then tap GenAlpaca Bug Reporter in the Add-ons section.

Desktop: Click the extension icon in the toolbar.

Then tap/click Capture Screenshot.

3

Annotate the screenshot

Use the tools to highlight the problem:

  • Draw — freehand drawing to circle/underline issues
  • Arrow — point at specific elements
  • Text — add labels or notes
  • Rectangle — box around an area

You can change the color, undo, or clear annotations. On mobile, use your finger to draw.

4

Describe the bug and submit

Tap Next, fill in your name, email, and describe what's wrong. Then tap Submit Bug Report.

What happens after you submit?

  • You'll get a confirmation email that your report was received
  • Our automated system analyzes your screenshot and description
  • It attempts to fix the bug and deploy the fix automatically
  • You'll get a follow-up email with what was fixed (or if it needs human attention)
  • Small problems usually take 2-5 minutes
Note: This extension works on Firefox for Android and Firefox desktop. For Chrome, use the Chrome version. Your name and email are saved locally so you don't have to re-enter them each time.
