AlpacApps Bug Reporter

Chrome extension for reporting bugs with annotated screenshots

Installation

1

Download the extension

Download and unzip the extension folder to your computer.

Download from GitHub

After unzipping, the extension is in the bug-reporter-extension folder.

2

Open Chrome Extensions page

Type chrome://extensions/ in your Chrome address bar and press Enter.

3

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the page.

4

Load the extension

Click Load unpacked (top-left), then navigate to and select the bug-reporter-extension folder from the download.

5

Pin the extension

Click the puzzle piece icon in Chrome's toolbar, then click the pin icon next to AlpacApps Bug Reporter so it's always visible.

How to Use

1

Navigate to the page with the bug

Go to the page where you see the issue.

2

Click the extension icon

Click the AlpacApps Bug Reporter icon in your toolbar, then click Capture Screenshot.

3

Annotate the screenshot

Use the tools to highlight the problem:

  • Draw — freehand drawing to circle/underline issues
  • Arrow — point at specific elements
  • Text — add labels or notes
  • Rectangle — box around an area

You can change the color, undo, or clear annotations.

4

Describe the bug and submit

Click Next, fill in your name, email, and describe what's wrong. Then click Submit Bug Report.

What happens after you submit?

  • You'll get a confirmation email that your report was received
  • Our automated system analyzes your screenshot and description
  • It attempts to fix the bug and deploy the fix automatically
  • You'll get a follow-up email with what was fixed (or if it needs human attention)
  • The whole process usually takes 2-5 minutes
Note: This extension only works on Chrome (or Chromium-based browsers like Edge, Brave, Arc). Your name and email are saved locally so you don't have to re-enter them each time.
